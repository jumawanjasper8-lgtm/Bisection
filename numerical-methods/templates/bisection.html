Bisection Method

A robust bracketing algorithm for finding roots of continuous functions.

Theory

Mathematical Discussion

The Bisection Method is one of the simplest and most reliable root-finding algorithms in numerical analysis. It is based on the Intermediate Value Theorem, which states:

Intermediate Value Theorem: If $f$ is continuous on $[a, b]$ and $f(a) \cdot f(b) < 0$, then there exists at least one root $c \in (a, b)$ such that $f(c) = 0$.

Algorithm

  1. Choose $a$ and $b$ such that $f(a) \cdot f(b) < 0$.
  2. Compute the midpoint: $c = \dfrac{a + b}{2}$
  3. If $f(c) = 0$ or $\dfrac{b - a}{2} < \varepsilon$, then $c$ is the root. Stop.
  4. If $f(a) \cdot f(c) < 0$, the root lies in $[a, c]$: set $b = c$.
  5. Otherwise, the root lies in $[c, b]$: set $a = c$.
  6. Return to Step 2.

Error Analysis

After $n$ iterations, the maximum possible error is:

$$E_n = \frac{b - a}{2^n}$$

The number of iterations needed to achieve a tolerance $\varepsilon$ is:

$$n \geq \left\lceil \log_2\!\left(\frac{b - a}{\varepsilon}\right) \right\rceil$$

Convergence

The bisection method converges linearly with a convergence rate of $\frac{1}{2}$ per iteration. While slower than Newton-Raphson or secant methods, it is guaranteed to converge as long as the initial bracket $[a, b]$ contains a root and $f$ is continuous.

Advantages & Limitations

✓ Advantages

  • Always converges (given a valid bracket)
  • Simple to implement and understand
  • Does not require derivatives

✗ Limitations

  • Requires a bracketing interval $[a, b]$
  • Slow linear convergence
  • Finds only one root per bracket
Examples

Worked Examples

Example 1 Find a root of \(f(x)=x^3-x-2\) on \([1,2]\)

Step 0: Verify the bracket

\(f(1)=1-1-2=-2<0\)

\(f(2)=8-2-2=4>0\)

Since \(f(1)\cdot f(2)=(-2)(4)=-8<0\), a root exists in \([1,2]\). ✓
